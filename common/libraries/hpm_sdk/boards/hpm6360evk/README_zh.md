## 概述


- 控制台串口参数：

    - 波特率：115200
    - 数据位：8位
    - 校验位：无
    - 停止位：1
    - 流控位：无
    - 数据编码方式：ASCII


## 引脚描述

(lab_hpm6360_evk_board)=

- SPI引脚：

| 功能 | 位置 |
| ---- | -------- |
| SPI3.CSN    | J28[24] |
| SPI3.SCLK   | J28[23] |
| SPI3.MISO   | J28[21] |
| SPI3.MOSI   | J28[19] |

- I2C引脚：

| 功能 | 位置 |
| ---- | -------- |
